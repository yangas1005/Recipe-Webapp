# Recipe-Webapp

Tony's personal cookbook app.

## Contents
- `index.html` — cookbook app homepage
- `recipes/` — recipe detail pages
- `recipe-images/` — locally stored recipe cover images

## Publish with GitHub Pages
1. Push this folder to the `Recipe-Webapp` repo
2. In GitHub repo settings, enable Pages from the main branch / root
3. The site should publish to:
   `https://<your-github-username>.github.io/Recipe-Webapp/`
